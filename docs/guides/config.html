Guía: Configuración y parámetros

Los modelos de InsideForest aceptan numerosos parámetros que controlan el número de árboles, la cantidad de clústeres y la estrategia de búsqueda.


InsideForestClassifier(
    n_estimators=100,
    n_clusters=10,
    auto_fast=False,
    rf_params=None,
    tree_params=None
)

Parámetros destacados

  • n_estimators: número de árboles en el bosque base.
  • n_clusters: cantidad de segmentos a priorizar.
  • auto_fast: activa ajustes automáticos para acelerar el entrenamiento.
  • rf_params y tree_params: diccionarios con opciones avanzadas para RandomForest y árboles individuales.

Estrategias de búsqueda

Se puede utilizar GridSearchCV o RandomizedSearchCV para optimizar los parámetros. Revisa los tutoriales para ver un ejemplo.
