Guía: Interpretación de resultados

InsideForest busca ofrecer interpretaciones claras para cada segmento encontrado.

Importancia de variables


importances = clf.feature_importances_
for feat, imp in zip(X.columns, importances):
    print(feat, round(imp, 3))

Descripciones de segmentos


descriptions = clf.describe_segments(top_k=3)
for desc in descriptions:
    print(desc["description"])

Cada descripción resume las condiciones que definen un grupo de instancias y puede utilizarse para generar reportes.

Visualizaciones

Puedes exportar los resultados a formatos como CSV o JSON y usar librerías de visualización (ej. plotly) para explorar interactivamente las regiones.
