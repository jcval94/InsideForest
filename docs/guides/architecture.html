Guía: Arquitectura interna

InsideForest se compone de varios módulos que interactúan para ofrecer interpretabilidad desde la fase de entrenamiento.

Módulos principales

| Módulo | Descripción |

|--------|-------------|

| trees | Construye y analiza los árboles de decisión subyacentes. |

| regions | Define y prioriza intersecciones de ramas para formar segmentos. |

| descrip | Genera descripciones legibles de cada región. |

| cluster_selector | Selecciona clústeres significativos empleando criterios supervisados. |

Flujo de datos

1. Entrenamiento: se ajusta un bosque de árboles de decisión sobre los datos etiquetados.

2. Extracción de ramas: cada árbol se recorre para identificar reglas de decisión.

3. Combinación: las ramas se combinan para formar regiones con características similares.

4. Selección: se priorizan las regiones más informativas con respecto a las etiquetas.

5. Descripción: se sintetizan reglas en lenguaje natural o representaciones tabulares.

Este diseño modular facilita extender o reemplazar componentes sin alterar el resto del sistema.
