Receta avanzada: Uso en conjuntos de datos masivos

Para escalar InsideForest a millones de registros se recomiendan las siguientes estrategias:

1. <code>auto_fast=True</code>: reduce automáticamente el número de ramas exploradas.

2. Submuestreo: utiliza una muestra representativa para entrenar el bosque y luego aplica el modelo al conjunto completo.

3. Paralelismo: ajusta n_jobs para aprovechar múltiples núcleos.

4. Reducción de características: habilita auto_feature_reduce o aplica técnicas como PCA antes del entrenamiento.


clf = InsideForestClassifier(auto_fast=True, n_jobs=-1, auto_feature_reduce=20)
clf.fit(X_big, y_big)

Estas recomendaciones ayudan a mantener tiempos de entrenamiento razonables sin sacrificar demasiada información.
