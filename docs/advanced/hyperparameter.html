Receta avanzada: Ajuste fino de hiperparámetros

Optimizar los parámetros de InsideForest permite mejorar la calidad de los segmentos y la precisión del modelo.


from sklearn.model_selection import GridSearchCV
from insideforest import InsideForestClassifier

param_grid = {
    "n_clusters": [5, 10, 15],
    "n_estimators": [100, 200],
    "auto_fast": [True, False]
}

grid = GridSearchCV(InsideForestClassifier(), param_grid, cv=3)
grid.fit(X, y)

print("Mejor configuración:", grid.best_params_)

Considera utilizar RandomizedSearchCV para espacios de búsqueda más amplios.
