ClusterSelector

Provides strategies to consolidate clusters generated by InsideForest according to different goals (balance, probability, quota).

Signature

ClusterSelector(method="select_clusters", **kwargs)

Available strategies

  • select_clusters: direct selection based on rules.
  • menu: maximizes an information objective.
  • max_prob_clusters: prioritizes higher probability segments.
  • balance_lists_n_clusters: enforces similar cluster sizes.
  • match_class_distribution: replicates the original class distribution.

Usage

from InsideForest import ClusterSelector

selector = ClusterSelector("match_class_distribution")
adjusted = selector.select(menu_structure, y)

The selector is typically used inside the InsideForest wrappers, but you can apply it manually to the menu_ structure as well.
