ClusterSelector

Permite elegir la estrategia de consolidación de clústeres según objetivos de negocio o métricas específicas.

Estrategias

  • select_clusters: selección directa.
  • menu: optimiza un objetivo de información.
  • max_prob_clusters: prioriza regiones con mayor probabilidad.
  • balance_lists_n_clusters: iguala tamaños de clúster.
  • match_class_distribution: mantiene la distribución original de clases.

Ejemplo

selector = ClusterSelector("menu", quota=0.2)
resultado = selector.select(menu_structure, y)

Se usa internamente por los envoltorios, pero puedes aplicarlo manualmente sobre menu_ para personalizar resultados.
