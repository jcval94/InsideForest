Labels

Convierte las regiones priorizadas en reportes y métricas listos para compartir.

Funciones

  • generate_descriptions(region_df, feature_names): textos descriptivos por clúster.
  • categorize_conditions(descriptions): agrupa condiciones por tipo.
  • summaries(clusterized): métricas de cobertura y pureza.
  • custom_round(number): redondeo contextual para reportes legibles.
  • get_intervals(interval_df): redacta intervalos a partir de los límites de cada rama.
  • get_labels(range_dataframes, df, target_var, max_labels=9, num_branches=10): obtiene descripciones, promedios del objetivo y subconjuntos completos para las ramas más representativas.

Ejemplo

from InsideForest import Labels

labels = Labels()
resumen = labels.summaries(clusteres)
descripciones = labels.generate_descriptions(descriptivo, feature_names)

resumenes_ramas = labels.get_labels(
    range_dataframes=rangos_priorizados,
    df=df,
    target_var="target",
    max_labels=5,
)

for rama in resumenes_ramas:
    for descripcion, (score, poblacion) in rama.items():
        media_objetivo, conteo = score
        print(f"{descripcion} → media={media_objetivo:.3f}, tamaño={conteo}")
        print(poblacion.head())

Los auxiliares de intervalos reutilizan custom_round para homogeneizar unidades y resaltar dígitos relevantes. Las descripciones y poblaciones resultantes se pueden exportar a CSV o Markdown y complementan los atributos df_clusters_description_ y df_datos_explain_ (disponibles en los envoltorios cuando activas get_detail=True). Consulta también reproducibilidad para documentar versiones.
