Metadata

Centralizes metadata generated during training for auditability and traceability.

Available information

  • Feature masks used (feature_mask).
  • Automatically applied parameters (fast_params).
  • Cluster size and purity statistics.

Example

from InsideForest import Metadata

meta = Metadata(clf)
report = meta.to_dict()

Useful for meeting regulatory requirements and documenting experiments alongside Reproducibility.
