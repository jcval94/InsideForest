Models

Herramientas para comparar modelos base, visualizar importancias y analizar desempeño.

Funciones

  • compare_models(models, X, y): compara métricas entre modelos.
  • plot_importances(model, top_n=20): grafica importancias.
  • confusion_matrix(labels, y): matriz de confusión para clústeres.

Ejemplo

models = Models()
reporte = models.compare_models([
    ("baseline_rf", baseline_rf),
    ("insideforest", clf.model_),
], X_test, y_test)
