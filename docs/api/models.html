Models

Utilities to evaluate and compare models associated with InsideForest.

Key functions

  • compare_models(models, X, y): compare metrics across different base models.
  • plot_importances(model, top_n=20): visualize cumulative importances.
  • confusion_matrix(labels, y): generate confusion matrices for clusters.

Example

from InsideForest import Models

models = Models()
summary = models.compare_models([
    ("baseline_rf", baseline_rf),
    ("insideforest", clf.model_),
], X_test, y_test)

Use these outputs alongside Metadata and the reports from Labels.
