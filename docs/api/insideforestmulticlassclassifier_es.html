InsideForestMulticlassClassifier (deprecado)

Usa InsideForestClassRegionClusterer en código nuevo. Este alias de compatibilidad emite FutureWarning, existe para migrar modelos serializados e imports anteriores y se retirará en 0.5.0.

from InsideForest import InsideForestClassRegionClusterer

modelo = InsideForestClassRegionClusterer(...).fit(X, y)
cluster_ids = modelo.predict(X_nuevo)
asignaciones = modelo.assign_regions(X_nuevo)

El estimador canónico hace clustering regional supervisado: no predice clases, no expone predict_proba y no utiliza fallback al bosque. Usa regions_for_class, ambiguous_regions y class_coverage_report para el análisis por clase.
