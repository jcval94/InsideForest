InsideForestRegionClusterer

Clusterer regional supervisado canónico. El bosque genera ramas; predict devuelve IDs de cluster y score devuelve adjusted mutual information.

from InsideForest import InsideForestRegionClusterer

modelo = InsideForestRegionClusterer(rf_params={"random_state": 42})
clusters = modelo.fit_predict(X_train, y_train)
asignaciones = modelo.assign_regions(X_test)
calidad = modelo.region_quality_report(X_test, y_test)

Contrato de salida

  • predict(X) devuelve IDs de cluster regional, nunca clases.
  • Las filas fuera de toda región reciben -1.
  • transform(X) devuelve la matriz muestra por región.
  • score(X, y) es adjusted mutual information (AMI), incluido -1.
  • regions_, region_metrics_, labels_ y raw_regions_ contienen los artefactos ajustados.

forest_ y feature_importances_ describen exclusivamente el bosque generador. InsideForestClassifier queda deprecado y sólo sirve para migración.
