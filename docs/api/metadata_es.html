Metadata

Proporciona un resumen estructurado de los metadatos generados durante el entrenamiento de InsideForest.

Incluye

  • feature_mask: variables retenidas tras la reducción automática.
  • fast_params: presets de rendimiento aplicados.
  • Estadísticas de clústeres: tamaño, pureza y cobertura.

Ejemplo

meta = Metadata(clf)
info = meta.to_dict()

Usa esta información para auditorías y documentación junto con la guía de reproducibilidad.
