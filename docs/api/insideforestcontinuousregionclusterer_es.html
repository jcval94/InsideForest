InsideForestContinuousRegionClusterer

Clusterer supervisado canónico para objetivos continuos. Un RandomForestRegressor genera hojas físicas; predict devuelve IDs regionales y score devuelve varianza explicada del objetivo (η²), incluido el cluster -1.

from InsideForest import InsideForestContinuousRegionClusterer

modelo = InsideForestContinuousRegionClusterer(
    rf_params={"n_estimators": 50, "max_depth": 6, "random_state": 42},
    leaf_percentile=90,
    min_support=3,
    branch_aggregation="none",
)
clusters = modelo.fit_predict(X_train, y_train)
asignaciones = modelo.assign_regions(X_test)
calidad = modelo.region_quality_report(X_test, y_test)

Cada hoja se puntúa una vez por cobertura, reducción de varianza y separación de la media global. Los empates se resuelven por menor desviación, mayor soporte y menor ID.

predict nunca devuelve estimaciones numéricas; una fila sin región recibe -1. forest_ e feature_importances_ describen exclusivamente el generador de ramas.
