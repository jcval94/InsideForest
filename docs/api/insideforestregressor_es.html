InsideForestRegressor (deprecado)

Usa InsideForestContinuousRegionClusterer en código nuevo. Este nombre emite FutureWarning, acepta parámetros históricos y conserva temporalmente el R² del bosque en score.

La clase canónica devuelve IDs, -1 explícito para filas sin región y η² en score. method, n_clusters, include_summary_cluster, divide y balance_clusters quedan sólo para migración.
