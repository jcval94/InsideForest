InsideForestClassifier

Firma

class InsideForestClassifier(...):
    def __init__(self, n_estimators=200, max_depth=None, random_state=None, ...): ...

Métodos

  • fit(X, y): entrena y extrae regiones.
  • predict(X): asigna clases.
  • score(X, y): evaluación.
  • save(path) / load(path): persistencia.

Ejemplo

import matplotlib.pyplot as plt
from InsideForest import InsideForestClassifier
clf = InsideForestClassifier(random_state=0).fit(X, y)
yhat = clf.predict(X)
plt.scatter(y, yhat)
plt.show()
