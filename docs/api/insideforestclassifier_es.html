InsideForestClassifier (deprecado)

Usa InsideForestRegionClusterer en código nuevo. Esta clase emite FutureWarning y sólo existe para cargar o migrar flujos históricos.

El alias conserva su score anterior de accuracy del bosque. El clusterer canónico devuelve IDs regionales en predict, -1 explícito para filas sin región y AMI en score.

# Migración
from InsideForest import InsideForestRegionClusterer

modelo = InsideForestRegionClusterer(...).fit(X, y)
cluster_ids = modelo.predict(X_nuevo)
detalles = modelo.assign_regions(X_nuevo)

No uses esta página como flujo nuevo de clasificación. Los clusterers regionales no ofrecen predicción de clases ni predict_proba.
