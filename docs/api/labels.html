Labels

Generates reports, metrics, and descriptions for the regions produced by Regions.

Functions

  • generate_descriptions(region_df, feature_names): produces short text per cluster.
  • categorize_conditions(descriptions): groups conditions by type.
  • summaries(clusterized): coverage, purity, and size metrics.

Example

from InsideForest import Labels

labels = Labels()
summary = labels.summaries(clusterized)
descriptions = labels.generate_descriptions(descriptive, feature_names)

These reports can be exported to CSV or Markdown and complement the describe_clusters methods in the main wrappers.
