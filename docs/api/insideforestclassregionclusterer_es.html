InsideForestClassRegionClusterer

Extrae regiones enriquecidas por clase. El bosque sólo genera ramas; no existe predicción de clases ni fallback.

from InsideForest import InsideForestClassRegionClusterer

modelo = InsideForestClassRegionClusterer(leaf_percentile=95, min_support=2)
clusters = modelo.fit_predict(X_train, y_train)
calidad = modelo.region_quality_report(X_test, y_test)

Cada hoja física se asocia únicamente con la clase que maximiza el objetivo de pureza, lift y cobertura; por eso produce como máximo una región final. Los solapamientos se resuelven por score, menor entropía, mayor soporte y menor ID.

classes_ es metadato de las distribuciones, no el espacio de salida de predict. Las filas no cubiertas reciben -1 y no existe fallback. La agregación IoU no es obligatoria: el valor soportado inicial es branch_aggregation="none".
