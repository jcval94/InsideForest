Automatiza pipelines de asignación regional

Integra InsideForest 0.4.3 en un flujo repetible con entradas explícitas, diagnósticos de calidad regional y persistencia.

Paso 1: preparar la estructura

mkdir -p pipelines artifacts

Paso 2: definir el script

# pipelines/daily_pipeline.py
from pathlib import Path
import pandas as pd
from InsideForest import InsideForestRegionClusterer

OBJETIVO = "target"
train = pd.read_parquet("data/train.parquet")
nuevo = pd.read_parquet("data/new.parquet")

modelo = InsideForestRegionClusterer(
    rf_params={"random_state": 42},
    seed=42,
    auto_fast=True,
    auto_feature_reduce=True,
)
modelo.fit(train.drop(columns=OBJETIVO), train[OBJETIVO])

asignaciones = modelo.assign_regions(nuevo)
calidad = modelo.region_quality_report(nuevo)

Path("artifacts").mkdir(exist_ok=True)
asignaciones.to_parquet("artifacts/asignaciones.parquet")
modelo.save("artifacts/insideforest.joblib")

La tabla de asignaciones registra el cluster -1 para filas sin región. Conserva también el reporte de calidad si necesitas auditar la cobertura histórica.

Paso 3: validar en CI

python -m pytest tests -q
python pipelines/daily_pipeline.py

Registra dependencias con Reproducibilidad y comunica resultados siguiendo la guía de Interpretación.
