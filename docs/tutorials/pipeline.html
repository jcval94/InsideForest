Tutorial: Integración con pipelines de scikit-learn

InsideForest puede integrarse en un Pipeline de sklearn para combinar preprocesamiento, selección de características y entrenamiento.


from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import GridSearchCV
from insideforest import InsideForestClassifier

pipe = Pipeline([
    ("scaler", StandardScaler()),
    ("model", InsideForestClassifier())
])

param_grid = {
    "model__n_clusters": [5, 10],
    "model__n_estimators": [100, 200]
}

grid = GridSearchCV(pipe, param_grid, cv=3)
grid.fit(X, y)

print("Mejores parámetros:", grid.best_params_)

Este enfoque facilita el ajuste fino de hiperparámetros y la reproducibilidad de experimentos.
