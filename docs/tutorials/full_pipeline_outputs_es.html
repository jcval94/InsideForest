Salidas del pipeline completo

Este tutorial documenta las salidas intermedias que conviene guardar cuando ejecutas InsideForest con get_detail=True. El objetivo es que cada etapa deje evidencia en tablas reproducibles: dimensiones, columnas clave y una muestra de hasta 10 registros.

Dependencias opcionales

pip install InsideForest==0.3.9
pip install MetaCraft==2025.10.6

Usa una función pequeña para imprimir siempre la misma ficha de auditoría. Si el DataFrame tiene 10 o más registros, muestra head(10); si tiene menos, muestra todos y conserva el shape.

import pandas as pd

pd.set_option("display.max_columns", None)

def audit_df(name, df, n=10):
    print(f"{name}: shape={df.shape}")
    print("columns:", df.columns.tolist())
    return df.head(n) if len(df) >= n else df

Iris: fit, etiquetas y regiones

from InsideForest import InsideForest
from sklearn.datasets import load_iris

iris = load_iris()
df = pd.DataFrame(iris.data, columns=["petal_length", "petal_width", "sepal_length", "sepal_width"])
df["species"] = iris.target.astype(float)

in_f = InsideForest(
    rf_params={"random_state": 15},
    tree_params={"lang": "py", "n_sample_multiplier": 0.05, "ef_sample_multiplier": 10},
    leaf_percentile=95,
    low_leaf_fraction=0.25,
    get_detail=True,
    var_obj="species",
)
in_f.fit(df)

pred_labels = in_f.predict(df.drop(columns=["species"]))
training_labels = in_f.labels_

Ejemplo de salida local: El grupo A tiene una efectividad de 0.28 frente a 0.72 en el grupo B, con una diferencia de 0.44 sobre Sobrevivió. Las variables que distinguen al grupo con menor efectividad son Sexo y Clase del boleto; el grupo de mayor efectividad se diferencia por Sexo y Tarifa. Esta comparación debe interpretarse como hipótesis descriptiva, no como recomendación directa, porque incluye atributos sensibles y proxies socioeconómicos.

Para reportes reproducibles, guarda junto al notebook las tablas df_clusters_description_, df_datos_explain, frontiers, metadata.df, top_experiments, df_exper y df_meta_sub. Así cualquier lector puede reconstruir la hipótesis desde los datos intermedios.
