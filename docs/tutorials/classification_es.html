Clustering regional guiado por clase

Encuentra regiones interpretables de Iris enriquecidas para una clase. Este flujo no predice especies: asigna IDs de clusters regionales.

Ajusta el clusterer

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from InsideForest import InsideForestClassRegionClusterer

X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, stratify=y, random_state=15
)
modelo = InsideForestClassRegionClusterer(
    rf_params={"n_estimators": 40, "max_depth": 5, "random_state": 15},
    leaf_percentile=90,
    min_support=2,
    branch_aggregation="none",
    random_state=15,
)
clusters_train = modelo.fit_predict(X_train, y_train)

Asigna y evalúa holdout

clusters_test = modelo.predict(X_test)
asignaciones = modelo.assign_regions(X_test)
reporte = modelo.region_quality_report(X_test, y_test)

Revisa coverage, unmatched_rate, ami, cluster_purity y las filas con source="unmatched". Ninguna predicción de clase rellena esas filas.

Interpreta las regiones

top = modelo.explain_regions(top_n=10)
clase_cero = modelo.regions_for_class(0, top_n=5)
ambiguas = modelo.ambiguous_regions(top_n=10)
cobertura_por_clase = modelo.class_coverage_report()

Cada fila de regions_ representa una hoja física seleccionada y conserva clase objetivo, distribución completa, lift, entropía, margen, soporte y límites geométricos.
