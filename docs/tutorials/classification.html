Tutorial: Clasificación supervisada básica

En este tutorial aprenderás a entrenar un InsideForestClassifier y a interpretar los segmentos resultantes.

1. Preparar los datos

Usaremos el dataset de vinos de sklearn, que incluye 13 características químicas por muestra.


from sklearn.datasets import load_wine
from sklearn.model_selection import train_test_split
import pandas as pd

X, y = load_wine(return_X_y=True, as_frame=True)
X_tr, X_te, y_tr, y_te = train_test_split(X, y, random_state=0)

2. Entrenar el modelo


from insideforest import InsideForestClassifier

clf = InsideForestClassifier(n_clusters=5, auto_fast=True)
clf.fit(X_tr, y_tr)

3. Interpretar segmentos

El modelo genera segmentos que resumen las reglas de decisión más relevantes.


segments = clf.get_segments()
print(pd.DataFrame(segments).head())

Cada fila representa una región en el espacio de características y puede describirse como una regla legible.

4. Evaluación


print("Precisión:", clf.score(X_te, y_te))

5. Variaciones sugeridas

  • Cambiar n_clusters para ajustar el número de segmentos.
  • Usar auto_feature_reduce=True para datasets con muchas variables.
  • Habilitar rf_params personalizados para controlar el bosque subyacente.
