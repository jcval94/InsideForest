Tutorial: Regresión supervisada

InsideForest también permite resolver problemas de regresión mediante InsideForestRegressor.

1. Dataset de ejemplo

Usaremos el dataset de diabetes de sklearn, que contiene 10 variables numéricas por paciente.


from sklearn.datasets import load_diabetes

X, y = load_diabetes(return_X_y=True)

2. Entrenar el modelo


from insideforest import InsideForestRegressor

reg = InsideForestRegressor(n_estimators=200)
reg.fit(X, y)

3. Predicción


preds = reg.predict(X[:5])
print(preds)

4. Evaluación rápida


from sklearn.metrics import r2_score
print("R2:", r2_score(y[:5], preds))

5. Consejos

  • Para datasets grandes considera auto_fast=True.
  • Ajusta n_estimators para equilibrar precisión y tiempo de entrenamiento.
