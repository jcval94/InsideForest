Regression tutorial

Model continuous values with InsideForest and explain each resulting region.

Goal

Predict diabetes progression using the classic Diabetes dataset included in scikit-learn.

Step 1: load data

from sklearn.datasets import load_diabetes

X, y = load_diabetes(return_X_y=True)

Step 2: train the regressor

from InsideForest import InsideForestRegressor

regr = InsideForestRegressor(auto_feature_reduce=True, auto_fast=True, get_detail=True)
regr.fit(X, y)
region_labels = regr.predict(X)
r2 = regr.score(X, y)

Step 3: analyze results

  • region_labels contains interpretable region labels; continuous estimates come from the fitted random forest with the same feature frame used during training.
  • Inspect regr.df_clusters_description_ and regr.df_datos_explain_ for rules and metrics.
  • Identify segments with high residuals for corrective actions.
  • Save the model with regr.save("models/housing.joblib").

Step 4: validate on multiple regression datasets

python experiments/validate_regression_regions.py --profile quick

The latest local quick run reports median test rule coverage of 0.9008, target-spread reduction of 0.5450, and region-mean RMSE lift of 0.1537. Full CSV outputs and the README are stored in experiments/results/regression_region_validation/.
