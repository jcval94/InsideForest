Regression tutorial

Model continuous values with InsideForest and explain each resulting region.

Goal

Predict housing prices using the California Housing dataset.

Step 1: load data

from sklearn.datasets import fetch_california_housing

data = fetch_california_housing()
X, y = data.data, data.target

Step 2: train the regressor

from InsideForest import InsideForestRegressor

regr = InsideForestRegressor(auto_feature_reduce=True, auto_fast=True)
regr.fit(X, y)

Step 3: analyze results

  • Generate rules with regr.describe_clusters(X, y).
  • Identify segments with high residuals for corrective actions.
  • Save the model with regr.save("models/housing.joblib").
