Tutorial de regresión

Predice valores continuos y explica segmentos con InsideForest.

Objetivo

Modelar precios de vivienda en el dataset California Housing.

Paso 1: cargar datos

from sklearn.datasets import fetch_california_housing

data = fetch_california_housing()
X, y = data.data, data.target

Paso 2: entrenar

from InsideForest import InsideForestRegressor

regr = InsideForestRegressor(auto_feature_reduce=True, auto_fast=True)
regr.fit(X, y)

Paso 3: interpretar

  • regr.describe_clusters(X, y) para reglas.
  • Localiza segmentos con error alto y genera acciones.
  • Guarda el modelo con regr.save("modelos/housing.joblib").
