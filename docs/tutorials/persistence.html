Tutorial: Guardado y carga de modelos

Es frecuente querer almacenar un modelo entrenado para reutilizarlo m치s adelante. InsideForest es compatible con joblib.


import joblib
from insideforest import InsideForestClassifier

clf = InsideForestClassifier()
clf.fit(X_tr, y_tr)

joblib.dump(clf, "modelo_iforest.pkl")

loaded = joblib.load("modelo_iforest.pkl")
print("Modelo cargado, precisi칩n:", loaded.score(X_te, y_te))
