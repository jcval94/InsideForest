Sección de desarrollo

Esta sección describe cómo instalar InsideForest en modo editable, ejecutar los tests y contribuir con nuevas funcionalidades.

Instalación local


git clone https://github.com/tu-organizacion/InsideForest.git
cd InsideForest
pip install -e .

Ejecutar tests y estilo


pre-commit run --all-files
pytest

Estructura del repositorio

  • src/: código fuente principal.
  • tests/: pruebas automatizadas.
  • data/: datasets de ejemplo.

Sigue las convenciones de estilo definidas por black y flake8 (si están configuradas en el proyecto) y procura añadir pruebas para cada nueva funcionalidad.
