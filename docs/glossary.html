Glosario

  • Rama: conjunto de reglas derivadas de un árbol de decisión.
  • Región: intersección de varias ramas que define un segmento del espacio de características.
  • Supervised clustering: técnica que utiliza etiquetas conocidas para guiar la formación de clústeres.
  • Segmento: grupo de instancias que comparte características similares y una etiqueta predominante.
