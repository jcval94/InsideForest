API rápida

InsideForest 0.4.3 ofrece tres clusterers regionales supervisados canónicos. Todos siguen fit/predict y predict devuelve IDs de regiones, no clases ni valores numéricos del objetivo.

Regiones supervisadas generales

from sklearn.datasets import load_iris
from InsideForest import InsideForestRegionClusterer

X, y = load_iris(return_X_y=True)
modelo = InsideForestRegionClusterer(
    rf_params={"n_estimators": 40, "random_state": 21}
)
labels = modelo.fit_predict(X, y)
asignaciones = modelo.assign_regions(X)
regiones = modelo.explain_regions(top_n=10)
calidad = modelo.region_quality_report(X, y)

labels y modelo.labels_ son IDs de cluster. score(X, y) devuelve AMI y una fila fuera de toda región recibe -1.

Regiones enriquecidas por clase

from InsideForest import InsideForestClassRegionClusterer

por_clase = InsideForestClassRegionClusterer(
    leaf_percentile=95,
    min_support=2,
    branch_aggregation="none",
    random_state=21,
)
cluster_ids = por_clase.fit_predict(X, y)
regiones_clase_0 = por_clase.regions_for_class(0, top_n=5)
ambiguas = por_clase.ambiguous_regions(top_n=10)
cobertura = por_clase.class_coverage_report()

Cada hoja seleccionada tiene una sola region_target_class y conserva su class_distribution completa. classes_ es metadato, no la salida de predict.

Asignación y filas sin región

assign_regions(X) devuelve el cluster, la clase objetivo de la región, score, distribución, lift, entropía, margen y todas las coincidencias. source sólo puede ser region o unmatched. No existen fallback al bosque ni predict_proba.

Persistencia

modelo.save("region-clusterer.joblib")
cargado = InsideForestRegionClusterer.load("region-clusterer.joblib")
assert (cargado.predict(X) == modelo.predict(X)).all()

Se conservan bosque, regiones, distribuciones, métricas y asignaciones. Los modelos antiguos pueden migrarse, pero deben recalcular labels_ si antes almacenaba una tabla de métricas.

Regresión

from InsideForest import InsideForestContinuousRegionClusterer

continuo = InsideForestContinuousRegionClusterer(
    leaf_percentile=90,
    min_support=3,
    branch_aggregation="none",
)
region_ids = continuo.fit_predict(X, y_continuo)
eta_cuadrada = continuo.score(X, y_continuo)
detalles = continuo.assign_regions(X)

predict devuelve IDs y score es η², incluido -1. InsideForestRegressor queda deprecado y conserva R² únicamente para migración.
