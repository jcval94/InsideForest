API Rápida

Clasificador

import matplotlib.pyplot as plt
from InsideForest import InsideForestClassifier
clf = InsideForestClassifier(random_state=0).fit(X, y)
y_pred = clf.predict(X)
plt.scatter(y, y_pred)
plt.show()
clf.save("clf.joblib")
clf2 = InsideForestClassifier.load("clf.joblib")

Regresor

import matplotlib.pyplot as plt
from InsideForest import InsideForestRegressor
regr = InsideForestRegressor(random_state=0).fit(X, y)
y_pred = regr.predict(X)
plt.scatter(y, y_pred)
plt.show()
regr.save("reg.joblib")
regr2 = InsideForestRegressor.load("reg.joblib")

Balanceo de clústeres

Activa balance_clusters=True para ponderar automáticamente las clases y usar el selector de clústeres "menu" al trabajar con datos multiclase:

clf = InsideForestClassifier(balance_clusters=True).fit(X, y)
