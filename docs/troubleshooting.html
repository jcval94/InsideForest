Solución de problemas

Error de instalación

Asegúrate de tener una versión reciente de pip y setuptools:


pip install --upgrade pip setuptools

Modelos lentos

  • Activa auto_fast=True.
  • Reduce el número de características o registros.
  • Usa paralelismo con n_jobs.

Problemas durante el entrenamiento

Verifica que no existan valores nulos y que las etiquetas estén bien formadas. Si el problema persiste, abre un issue en el repositorio.
