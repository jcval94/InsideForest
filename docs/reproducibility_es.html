Reproducibilidad

  • Fija random_state para resultados deterministas.
  • Registra las versiones de paquetes incluyendo insideforest.
  • Semilla los RNG de Python y NumPy.
import random, numpy as np
from InsideForest import InsideForestClassifier
random.seed(0); np.random.seed(0)
clf = InsideForestClassifier(random_state=0)
