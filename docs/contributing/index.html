Contribuir

Tu ayuda es bienvenida. Antes de enviar cambios sigue estas recomendaciones.

Flujo de trabajo

1. Haz un fork del repositorio.

2. Crea una rama descriptiva para tu aporte.

3. Asegúrate de que pre-commit y pytest pasen sin errores.

4. Abre un Pull Request detallando los cambios.

Plantillas

Utiliza las plantillas de issues y PRs para mantener la consistencia.

Código de conducta

Todos los participantes deben respetar nuestro código de conducta si está disponible.
