Performance Tips

  • n_estimators: fewer trees speed up training.
  • n_jobs: use parallelism for heavy tree extraction.
  • Reduce memory by lowering max_depth or sampling data.

FAST options

auto_fast

Automatically chooses faster training parameters based on dataset size.

clf = InsideForestClassifier(auto_fast=True).fit(X, y)

auto_feature_reduce

Reduces the number of features before training. Useful when there are many columns.

clf = InsideForestClassifier(auto_feature_reduce=True).fit(X, y)

explicit_k_features

Locks the number of retained features when auto_feature_reduce is enabled.

clf = InsideForestClassifier(auto_feature_reduce=True,
                             explicit_k_features=20).fit(X, y)

fast_overrides

Overrides parts of the automatic FAST preset.

clf = InsideForestClassifier(auto_fast=True,
                             fast_overrides={"divide": 3}).fit(X, y)
