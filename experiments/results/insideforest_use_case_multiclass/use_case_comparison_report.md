# Archived classifier use-case report

This directory contains figures and tables produced by the old multiclass classifier example. They include class predictions, confusion matrices, confidence, and forest fallback, none of which are outputs of the current region clusterer.

Use `InsideForest/examples/InsideForest_Caso_de_Uso.ipynb` for the maintained examples and run `python experiments/validate_class_region_clusters.py --profile quick` for current quantitative validation.
